# Diwali_Sales_Analysis_Python

## Overview

This project analyzes Diwali sales data using Python to understand customer purchasing behavior and sales trends.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Analysis Performed

* Data Cleaning
* Missing Value Handling
* Gender-wise Analysis
* Age Group Analysis
* State-wise Analysis
* Marital Status Analysis
* Occupation Analysis
* Product Category Analysis

## Key Insights

* Female customers contributed more to sales.
* Customers aged 26–35 years were the most active buyers.
* Maharashtra, Karnataka, and Uttar Pradesh generated high sales.
* Certain product categories showed stronger customer demand.

## Files

* `Diwali_Sales_Analysis.ipynb` – Jupyter Notebook
* `Diwali Sales Data.csv` – Dataset
